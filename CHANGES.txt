===============
PyAMF Changelog
===============

This file contains information about the changes between the
different versions of PyAMF.

0.1.0 (unreleased)
------------------------------------------------------------
 - AMF0 and AMF3 encoders/decoders
 - Additional support for IExternalizable, ArrayCollection,
   ObjectProxy, ByteArray, RecordSet and basic RemoteObject
   support.
 - Remoting gateways for Twisted, WSGI, and Django
 - Authentication/setCredentials support
 - zlib compression support for ByteArray
 - Remoting client for httplib

2007-10-19
------------------------------------------------------------
 - Started project based on previous work done in the
   RTMPy (http://rtmpy.org) project.
